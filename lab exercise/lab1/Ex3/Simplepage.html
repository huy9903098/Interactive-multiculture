HTML: It is a language that is used to created website.

CSS: It is a stylesheet language used to determine how HTML element will be displayed on the web/screen.

Javascript: It is a scripting language is used to enhance web pages by making it interactive.

HTTP: It is a set of rules that allow users on the Web to exchange information with each other.

HTML5: This is the latest version of HTML, it can created a web application faster.

DHTML: It is a combination of HTML, Javascript, CSS which is used to create dynamic website.
